## ? Setup Checklist

### In Unity Editor:

1. **Create Close Skill Tree Button**
   - [ ] Right-click SkillsPanel ? UI ? Button
   - [ ] Name: "CloseButton"
   - [ ] Position: Top-right
   - [ ] Text: "Close" or "×"

2. **Create Back Button**
   - [ ] Right-click PreviewPanel ? UI ? Button
   - [ ] Name: "BackButton"
   - [ ] Position: Bottom-left or top-left
   - [ ] Text: "Back" or "?"

3. **Assign in SkillTreeUI_GridBased**
   - [ ] Select SkillsPanel
   - [ ] Find SkillTreeUI_GridBased component
   - [ ] Drag CloseButton to "Close Skill Tree Button" field
   - [ ] Drag BackButton to "Back Button" field

4. **Assign in SaveStationMenu**
   - [ ] Select SaveStationMenu object
   - [ ] Find SaveStationMenu component
   - [ ] Drag SkillsPanel to "Skill Tree UI Grid" field

5. **Test**
   - [ ] Press Play
   - [ ] Open save station
   - [ ] Click Skills
   - [ ] Click a skill card
   - [ ] Click Back ? Should show grid
   - [ ] Click Close ? Should show save menu
   - [ ] Test with Escape key
   - [ ] Test with gamepad

## ?? Common Mistakes to Avoid

? **Don't** assign the wrong button
   - Close button goes on SkillsPanel (main panel)
   - Back button goes on PreviewPanel (sub-panel)

? **Don't** forget to assign both buttons
   - Both are needed for full navigation

? **Don't** use onClick in Inspector
   - Scripts handle onClick automatically
   - Just assign the button reference

? **Don't** assign list UI to grid field
   - Use "Skill Tree UI" for SkillTreeUI (list)
   - Use "Skill Tree UI Grid" for SkillTreeUI_GridBased

? **Do** test both buttons
? **Do** test keyboard and gamepad
? **Do** verify event subscription

## ?? Pro Tips

### Tip 1: Button Icons
Use TextMeshPro to add icons:
```
Font: FontAwesome or similar icon font
Text: "" (Unicode left arrow)
Size: 24
```

### Tip 2: Hover Effects
Add transition to buttons:
```
Transition: Color Tint
Normal: White
Highlighted: Light Blue
Pressed: Dark Blue
```

### Tip 3: Sound Effects
Add AudioSource to buttons:
```
On Click: Play("UIClick")
On Back: Play("UIBack")
```
